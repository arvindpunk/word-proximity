Word Edit

        
        Short Description

        Guess a new word everyday based on how near or far each letter is.

        
        Long Description

        Word Edit is a play on the popular Word game genre.

        A new word to play every single day!

        In this game, each letter of the word is denoted by a specific shade of yellow, denoting how close or far away it is
        from the actual word.

        Green denoting you've found the correct letter.
        Four levels of yellow denote how far away the letter is from the actual one.
        - Brightest: actual letter is just one off!
        - Little less bright: actual letter is upto 3 places off
        - Lot less bright: actual letter is upto 5 places off
        - Darkest: actual letter is upto 7 places off
        Grey or no color denotes actual letter is more than 7 places off.
